# Contributing

Thanks for your interest in contributing to **Mindful Breathe**.

## Getting Started

1. Fork the repository
2. Create a feature branch from `main`
3. Install dependencies:

```bash
npm install
```

4. Start the app:

```bash
npx expo start
```

## Development Guidelines

- Keep changes focused and minimal
- Follow existing code style and folder structure
- Prefer clear naming over abbreviations
- Update docs when behavior or setup changes

## Before Opening a Pull Request

Run checks locally:

```bash
npm run lint
```

If relevant, also test on at least one target platform:

- iOS simulator/device
- Android emulator/device

## Pull Request Checklist

- Include a clear description of what changed and why
- Link related issues (if any)
- Add screenshots or recordings for UI changes
- Note any breaking changes or migration steps
- Confirm lint passes

## Commit Message Suggestions

Use short, descriptive commits. Common style:

- `feat: add preset quick actions`
- `fix: correct inhale hold transition`
- `docs: update setup instructions`
- `refactor: simplify timer state flow`

## Reporting Issues

When reporting a bug, include:

- Expected behavior
- Actual behavior
- Steps to reproduce
- Device/OS version
- App version

## Code of Conduct

Be respectful, constructive, and collaborative.
